miRNACNet: How it Works


To test your microRNA samples, simply head over to the 'Input Data' tab. From there, you will be able to input a CSV file containing microRNA data. Please note that our application only works with CSV files, so please make sure your data is formatted correctly. Our model is able to predict whether the microRNA samples provided are cancerous or non-cancerous, and will give a percentage that shows how confident the model is. Along with the results there will be different visualizations which will allow you to interpret the results more easily.

Getting Started


After browsing our front page, you can create an account with an email, username and password. After doing so, you will have access to our predictive model so that you can start inputting your own miRNA data samples. Please note that in order to access the input page or dashboard, you must have a registered account.

Input


After navigating to the input page, locate your file containing miRNA data. Remember that it must be a csv file that is correctly formatted. The first row should be the labels of each attribute (miRNA_ID, read_count, reads_per_million_miRNA_mapped, and cross-mapped). The other 1881 rows should be miRNA samples. Below is an example ofhow your file should be formatted:




Once you confirm that your data is correctly formatted, select what type of cancer your file contains. Then, select the type of machine learning model you want to predict on (SVM or Random Forest) and the training cancer type for the selected model (Breast, Kidney, Lung or Multiclass).

Results


Once you submit the input form, to the results section, where your prediction will be displayed, and a link to the details of the selected model will be shown.

Dashboard


As you continue to run predictions, your results will be saved for you to access at any time. Simply head to the Dashboard tab to access your account's dashboard.




From there, you will be able to access all of your previous results, making access to past predictions quite convenient.

Other Information


To learn more about each our models, head to the Models tab. From there you will also be able to access the page for each specific dataset.

If you wish to learn more about our project, head to the home page by clicking our logo at the top left. We hope you enjoy the information our website has to offer!
